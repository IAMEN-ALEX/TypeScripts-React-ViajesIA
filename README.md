# NotadeViajesIA

Proyecto de aplicación web de viajes con integración de IA.

## Características

- Login con diseño Glassmorphism
- Animaciones interactivas (Three.js)
- Base de datos SQLite
- Next.js 14+ App Router
- TypeScript + React    
- IA con OpenAI gemini 3.5  
- Dashboard con notas y viajes+IA

## Ejecución

```bash
npm install
npm run dev
```
